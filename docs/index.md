## アラーム機能

### アラームをセットする(10時00にセット)
```
!alaram 10:00
```
コマンドを実行したらその時間にBotが通話部屋に入りアラームを鳴らしてくれる。<br>
※セットした人が通話部屋に入っていない場合Botは来てくれない。
